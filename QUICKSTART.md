# 🚀 Quick Start Guide

Get BookBeta up and running in 5 minutes!

## Step 1: Install Dependencies

```bash
npm install
```

## Step 2: Add Your Chapters

Create markdown files in the `chapters/` folder:

**Example: `chapters/chapter-01.md`**
```markdown
---
title: "Chapter 1: The Beginning"
order: 1
---

Your chapter content here...
```

## Step 3: Start the Server

```bash
npm run dev
```

Open http://localhost:3000

## Step 4: Share with Readers

1. Visit http://localhost:3000/admin (Author Dashboard)
2. Copy the reader link shown on the dashboard
3. Share it with your beta readers

## Step 5: Collect Feedback

Readers can:
- Highlight any text
- Click Like/Dislike for quick feedback
- Leave detailed comments
- Suggest alternative wording

All feedback appears in your dashboard at `/admin`

## That's It!

You're ready to collect feedback on your writing.
